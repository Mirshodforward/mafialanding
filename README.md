# Mafia Online — Landing

Marketing landing page for **Mafia Online** (the game lives at
[game.mafiaonline.uz](https://game.mafiaonline.uz)). Built with **Astro 5**,
plain CSS, fully static, bilingual (UZ + RU), SEO/AEO‑optimised.

## Stack
- **Astro 5** — `output: 'static'`, deploys to Vercel as-is.
- **@astrojs/sitemap** — i18n sitemap (uz/ru) at `/sitemap-index.xml`.
- **Plain CSS** — dark "noir" theme in `src/styles/global.css`, scoped styles per component. No CSS framework.
- **sharp** (dev only) — cleans the master logo (`assets/Mafialogo.png`) and rasterises the PNG icons + OG image.

## Scripts
```bash
npm install        # install deps
npm run dev        # local dev server
npm run build      # static build → dist/
npm run preview    # preview the production build
npm run assets     # regenerate logo.png + icons + og-image.png from assets/Mafialogo.png
```

## Structure
```
src/
  layouts/Layout.astro      # SEO head (meta/OG/Twitter/canonical/hreflang/JSON-LD slot), fonts, scroll-reveal
  i18n/copy.ts              # all UZ + RU landing content + the 13 roles (single source of truth)
  i18n/blog.ts              # blog content + UI strings (5 posts × UZ/RU/EN) + SEO/schema helpers
  components/               # landing: Navbar, Hero, Roles … + blog: Blog{Header,Footer,Card,Body,CTA,IndexView,PostView}
  pages/
    index.astro             # UZ homepage  (/)
    ru/index.astro          # RU homepage  (/ru)
    blog/, ru/blog/, en/blog/   # blog index + [slug] posts (UZ / RU / EN)
assets/
  Mafialogo.png             # master brand logo (source — not deployed)
public/
  logo.png, *.png, og-image.png   # generated by `npm run assets`
  robots.txt, site.webmanifest
```

Landing components take a `lang` prop (`"uz" | "ru"`) and read from `src/i18n/copy.ts`,
so both languages share the exact same markup/design. The **blog** adds a third
language (`en`) and is fully data-driven from `src/i18n/blog.ts` — add a post there
and the UZ/RU/EN index, post pages, sitemap and hreflang links update automatically.

## SEO / AEO
- Per-page `<title>`, meta description, keyword set (UZ/RU/EN), canonical, hreflang (uz/ru/en/x-default).
- JSON-LD: `WebSite`, `VideoGame`, `Organization`, `FAQPage`; blog adds `Blog`, `BlogPosting`, `BreadcrumbList`.
- AEO: concise direct-answer line in the hero + question-format FAQ with schema.
- Blog: keyword-targeted long-form articles (how-to-play, video chat, 13 roles, strategy, game night) in 3 languages.
- `robots.txt` explicitly allows AI crawlers (GPTBot, ClaudeBot, PerplexityBot…).

## Editing
- **Copy / translations** → `src/i18n/copy.ts`.
- **Roles** → the `ROLES` array in `src/i18n/copy.ts` (mirrors the game's `src/constants/roles.ts`).
- **Play button target** → `PLAY_URL` in `src/i18n/copy.ts`.
- **Logo** → replace `assets/Mafialogo.png`, then run `npm run assets`.

## Deploy
Push to a Vercel project (framework auto-detected as Astro). Set the domain to
`mafiaonline.uz`. No env vars required.
